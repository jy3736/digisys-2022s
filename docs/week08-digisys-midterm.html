[實作問題 1]：邏輯閘層級(Gate Level) Verilog 電路實作。

  • 參考作業：第4周 課堂作業 1
  • 邏輯閘層級(Gate Level) Verilog 電路實作。(module : prob01)

  • 你的電路必須通過功能模擬測試 (Testbench)。 (module : prob01_tb)
`timescale 1ns / 1ps

module prob01_tb;

    reg a,b,c,d;
    wire f;
    wire x1,x2,x3,fx;
    reg err;

    assign x1 = a | b;
    assign x2 = ~a & c;
    assign x3 = x1 | ~x2;
    assign fx = ~d | x3;

    prob01 uut1(.a(a),.b(b),.c(c),.d(d),.f(f));
    
    integer i;
    initial begin
        {a,b,c,d,err} = 0;
        #100;
        for(i=0; i<16; i=i+1) begin
            err = 1'b0;
            {a,b,c,d} = i;
            #50;
            err = (f!=fx);
            #50;
        end
        #100;
        $finish;
    end

endmodule

[實作問題 2]：使用半加器 (half adder) 設計全加器 (full adder) + 測試程式。

  • 參考作業：第5周 課堂作業 1
  • 使用 Data Flow 1-bit 半加器 (module : hadd1)，實現 1-bit Structural Level 全加器。 (module : fadd1)
`timescale 1ns / 1ps

module hadd1(
    input a,
    input b,
    output s,
    output co
);

    assign {co,s} = a + b;
    
endmodule

  • 參考『實作問題 1』功能模擬測試 (Testbench)，設計對應的全加器測試程式。(module : fadd1_tb)

[實作問題 3]：3-bit 連波進位加法器 (Ripple Carry Adder) 電路實作。

  • 參考作業：第6周 課堂作業 1
  • 將原圖4-bit電路改成3-bit，用作考試參考用。
  • 使用 Data Flow 1-bit 全加器 (module : fadd1)，以 Structural Level Verilog 的方式實現「3-bit 連波進位加法器 」。(module : rcadd3)
`timescale 1ns / 1ps

module fadd1(
    input a,
    input b,
    input ci,
    output s,
    output co
    );
    
    assign {co,s} = a + b + ci;
    
endmodule
  • 參考『實作問題 1』功能模擬測試 (Testbench)，設計對應的3-bit加法器測試程式。(module : rcadd3_tb)
