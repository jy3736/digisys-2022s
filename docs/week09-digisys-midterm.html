[實作問題 1]：邏輯閘層級(Gate Level) Verilog 電路實作。

  • 參考作業：第4周 課堂作業 1
  • 邏輯閘層級(Gate Level) Verilog 電路實作。(module : prob01)

  • 你的電路必須通過功能模擬測試 (Testbench)。 (module : prob01_tb)
`timescale 1ns / 1ps

module prob01_tb;

    reg a,b,c,d;
    wire f;
    wire x1,x2,x3,fx;
    reg err;

    assign x1 = a | b;
    assign x2 = ~a & c;
    assign x3 = x1 | ~x2;
    assign fx = ~d | x3;

    prob01 uut1(.a(a),.b(b),.c(c),.d(d),.f(f));
    
    integer i;
    initial begin
        {a,b,c,d,err} = 0;
        #100;
        for(i=0; i<16; i=i+1) begin
            err = 1'b0;
            {a,b,c,d} = i;
            #50;
            err = (f!=fx);
            #50;
        end
        #100;
        $finish;
    end

endmodule

[實作問題 4]：選擇性檢測待測電路 4-bit ALU。

  • 參考作業：第7周 課堂作業 2
  • 參考『實作問題 1』功能模擬測試 (Testbench)，設計對應的4-bit ALU測試程式。(module : alu4_tb)
  • 本題是選擇性檢測『邏輯運算(&、|、^)』，其他功能檢測『不可』加入測試程式中。

Data Flow Modeling Toy ALU (module : alu4)

`timescale 1ns / 1ps

module alu4(
    input [3:0] a,
    input [3:0] b,
    input [2:0] op,
    output [3:0] dout
    );
    
    wire [3:0] add;
    wire [3:0] sub1;
    wire [3:0] sub2;
    wire [3:0] op_and;
    wire [3:0] op_or;
    wire [3:0] op_xor;
    
    assign add = a + b;
    assign sub1 = a - b;
    assign sub2 = b - a;
    assign op_and = a & b; 
    assign op_or = a | b; 
    assign op_xor = a ^ b;
    
    assign dout = (op==0)?0:
                  (op==1)?add: 
                  (op==2)?sub1: 
                  (op==3)?sub2: 
                  (op==4)?op_and: 
                  (op==5)?op_or: 
                  (op==6)?op_xor:7; 
    
endmodule。

完整模擬的資料量龐大，底下模擬波型圖看看就好，現場會要求展開局部波型來觀察。

[實作問題 5]：在 Vivado 中載入現有的 Verilog 電路並完成測試。

  • 參考作業：第7周 課堂作業 3
  • 先建立一個空的 Vivado 專案
  • 將 prob05.v 設計加入專案中
`timescale 1ns / 1ps

module prob05(
    input rst,
    input clk,
    output [3:0] dout
);

    reg [3:0] dat;
    
    assign dout = dat;
    
    always@(posedge clk) begin
        if (rst==0)
            dat = 0;
        else
            dat = dat + 1;
    end
  • 執行 RTL 電路分析
  • 將測試程式 prob05_tb.v 加入專案中
`timescale 1ns / 1ps

module prob05_tb;

    reg clk;
    reg rst;
    wire [3:0] dout;

    prob05 uut1(.rst(rst),.clk(clk),.dout(dout));

    always #50 clk = ~clk;

    initial begin
        {clk, rst} = 0;
        #100
        rst = 1'b1;
        #1600;
        rst = 1'b0;
        #300;
        rst = 1'b1;
        #1600;
        $finish;
    end

endmodule
  • 完成功能模擬測試
